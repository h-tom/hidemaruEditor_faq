[HMM0056A]

●複数のウィンドウ間を移動したい

openfile文でファイルを開くと、その後の処理は今開いたファイルに対して行われますが、元のファイルに戻るにはどうしたらよいのでしょうか？

既に開かれている秀丸エディタのウィンドウをアクティブにする(画面の一番手前に持ってくる)には、setactivehidemaru文を使います。

//------- サンプルマクロ -------//
// 元のファイルに戻る
#hwnd = hidemaruhandle( 0 ); //元ウィンドウのハンドルを取得する
openfile "c:\\hidemaru\\install.txt"; //別ファイルを開く

    //ここに c:\\hidemaru\\install.txt 上での処理が入る。

setactivehidemaru #hwnd; //元のウィンドウに戻る
endmacro;
// ---------------------------- //

※ hidemaruhandle()関数についてはマクロヘルプの秀丸エディタ管理の説明と「●setactivehidemaru文の引数」を参照して下さい。
