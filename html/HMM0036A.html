[HMM0036A]

●IMEのマクロ動作中切断方法

マクロの実行前後で、IMEの状態を同じにしておきたいのですが。

マクロの実行直後と終了直前にIMEの状態を調査して、希望の状態になるように設定します。次の例はマクロ実行中はIMEをOFFにし、マクロ実行後に、IMEのON/OFF状態を元の状態に戻します。

//------- サンプルマクロ -------//
#imeYN = imestate;
if( #imeYN ) imeswitch; // IME を OFF
//以下、マクロ本体
// ……ここからマクロ本体を書く

//終了直前
if( #imeYN != imestate ) imeswitch;
endmacro;
// ---------------------------- //
