[HME0007A]

●問い合わせ時の必要事項

問い合わせをするとき、どの様なことを書いておけばよいのでしょう？

まず [HME0006A] ●質問したい時にはどうするの？を一読して、他の方法で解決できないかどうか確かめてください。実際に問い合わせるときは、状況に応じて次のようなことを解る範囲で構いませんので、出来るだけ詳しく、具体的に書いておくと良いでしょう。

よく、「普通に・・・」と言う人がいますが、人によって「普通」は異なります。面倒に感じますが、手順などは詳細に説明しましょう。

  1. 不具合と考えられる場合

    1. 何をしたのか？
    2. どうなると思っていたのか？
    3. 実際にはどうなったのか？
    4. どの様な設定でも起きるのか？
    5. 発生する確率はどの程度か？

  2. 自分の希望する機能が、その存在の有無も含めて解らない場合

    1. 自分のやりたい事は何なのか？
    2. どの様な操作をして、その結果どうなるのがよいのか？

  3. 機能を追加して欲しい場合

    1. どの様な機能がほしいのか？
    2. 既にある機能ならどの様に拡張してほしいのか？
    3. 何処にどの様に表示するのか？
    4. ユーザーインターフェース (使い方) はどの様な物が良いか？

  4. 質問の内容と関わりなく、書いた方が早く解決できる可能性のある事柄

    1. 使用している秀丸エディタのバージョン
      タブモードで使用しているかどうかや、モニタのDPIもわかると良いかも
    2. Windows のバージョン (XP、Vista、7、8、10 など)
      Windows 10 の場合は、詳細なバージョンがわかる方がいいでしょう(1709や、1803，1809など)
    3. 自分の使用しているパソコンの機種名
      同じ機種を使用している人がいれば、何か情報があるかもしれません。
    4. ファイルシステム (FAT、FAT32、NTFS)
    5. 問題が起きる接続先がネットワークで共有している場合、サーバーのOS、アプリケーション (Samba など) とファイルシステムおよびそのヴァージョン
    6. 例外的に、特定の環境でのみ起きることが解っていれば、問題の起きる環境、起きない環境の違いを書く

1. の場合は、

  • 他のソフトウェアで同じことが起きないか？
  • この Q＆Aに同様の記載は無いか?
  • 自動起動マクロが有効な場合は、無効にすると変化があるか?

を合わせて確認した後、すぐに質問しないで、コミュニテックスのサポートフォーラムを一度確認してみて、既に報告されているようならば、修正を気長に待ましょう。
(サポートフォーラムに関しては、[HME0006A]●質問したい時にはどうするの？を参照)

2. 、3. については、他のアプリケーションで出来る機能なら、そのアプリケーションと機能名を記載すると解りやすいでしょう。

またどの様な場合も、[HME0008A]●発言タイトルの付け方は？を参考に、適切な題名を付けるようにしましょう。

但し、コミュニテックスなど多くの人の目に触れる場所に投稿する場合は、パスワード、自分の住所などは、絶対に書いてはいけません！
