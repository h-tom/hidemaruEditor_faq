[HMM0076A]

●重複行の削除

: 重複行の削除はできませんか？。

下記のマクロで可能です。

秀まるおのホームページ(サイトー企画)−秀丸エディタの変換モジュールライブラリに、 uniqの機能を持つ変換モジュールが登録されているので、それを使う方法もあります。

//------------------ uniq.mac -------------------------------------

disabledraw;
gofiletop;
while(1){
    selectline;
    $$line = gettext(seltopx,seltopy,selendx,selendy);
    if($$line == $$bline){
        selectline;
        delete;
    }
    else movetolineno 1,lineno + 1;
    if(lineno == linecount2) break;
    $$bline = $$line;
}
enabledraw;
endmacro;
